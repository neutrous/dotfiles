### HOWTO

Firstly, you should install [spacemacs][]

Then:

```
    $ git clone https://github.com/neutrous/dotfiles && cd dotfiles
    $ ln -s neutrous-tools ~/.emacs.d/private/
    $ cp -f dot-spacemacs ~/.spacemacs
```
